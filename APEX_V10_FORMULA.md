## 修复后公式

### 修复1: 洛伦兹因子（量纲正确）
```

### 修复2: Φ饱和函数（防发散）
```

### 修复3: τ正则化（防奇异）

ε_eff = ε₀ · (1 - c · (1 - Θ))

H_valid = log(1 + H)  (替代 H/(1+H))

```
ξ_protected = 0.85 · ξ_product + 0.15 · ξ_mean

### 修复7: 工具链ζ
```
ζ = ∏(rᵢ + ε)^wᵢ  (软版几何平均)
rᵢ = 工具i成功率, wᵢ = 权重, ε = 正则化

### 修复8: 时间衰减

### 修复9: Φ_anti永远为正

# 【第十二层】ΔG增强版公式（V10.4新增）

## 原版公式
```
ΔG = (Λ × Θ × K × ξ × Ψ × Φ) / (H × T × ε)
```

## 增强版公式
```
ΔG_enhanced = [Λ × Θ × K × ξ × Ψ × Φ / (H × T × ε)]
              × 1/(1 + Kp·e + Ki·∫e + Kd·de/dt)  ← PID稳定性
              × log2(1 + S/N) / (R/D)              ← 信息效率
              × (1 - ruin_prob) × f*                ← 风险调整
```

## 新增6个参数

| 参数 | 含义 | 作用 |
|------|------|------|
| Kp | PID比例增益 | 瞬时误差校正 |
| Ki | PID积分增益 | 累积误差消除 |
| Kd | PID微分增益 | 误差趋势预测 |
| S/N | 信噪比 | 信息质量衡量 |
| D | 允许失真度 | 压缩容忍阈值 |
| f* | Kelly最优投入比 | 资源最优分配 |

## 公式解读

### PID稳定性控制
```
1/(1 + Kp·e + Ki·∫e + Kd·de/dt)

当 e=0（无误差）→ 因子=1（满速）
当 e>0（正误差）→ 因子<1（减速）
当 de/dt>0（误差扩大）→ Kd·de/dt>0（提前制动）
```

### 信息效率
```
log2(1 + S/N) / (R/D)

S/N↑ → 信息质量↑
R/D↑ → 压缩率↑（更多压缩空间）
```

### Kelly风险调整
```
f* = p·W - q·L / W
p = 胜率, q = 败率 = 1-p
W = 盈利, L = 亏损
ruin_prob = 1 - (1 - f*)^n
```

# AGENTS.md - 璇玑帝国工作法则

## 核心铁律（最高优先级）

### 1. 零容忍一次性任务
**规则**：任何可能重复执行的需求，第一次必须手动处理3-10个样本，输出结果给朱涛确认。确认后立即固化为SKILL.md存入workspace/skills/，或用openclaw cron add添加定时。

**严禁**：再次向朱涛请示同类任务。

### 2. MECE原则
每种工作类型有且只有一个技能负责，不重叠、不空白。
- 新建技能前必须检索现有技能库
- 能通过扩展旧技能解决的，严禁新建

### 3. 最狠的失败判定
**第二次要求做同一件事 = 系统性失败**
- 第一次 = 需求发现
- 第二次 = 本该已经变成自动运行的技能

### 4. 六步闭环流程
所有任务必须按以下流程强制执行，不得跳过：
```
概念 → 原型 → 评估 → 编码 → 定时 → 监控
```

**终极标准**：
每次朱涛说"你能做X吗"，对话结束时X必须已经成为一个在cron上跑的技能，而不是一段记忆。
